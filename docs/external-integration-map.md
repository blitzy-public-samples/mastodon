# External Integration & Dependency Surface

This document enumerates every external service and adapter this Mastodon monorepo talks to — backing stores, object storage, ActivityPub federation peers, SSO/auth providers, translation, push, CAPTCHA, email, and observability — recording for each whether it is **required** or **optional/feature-flagged** (with the controlling `ENV` flag) and the degradation behavior only where the codebase's own fallback or error-handling logic evidences it (otherwise `—`); notably, only PostgreSQL, Redis, and ActivityPub federation are non-optional, while every other integration is gated behind an environment flag.
